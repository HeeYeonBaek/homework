# 첫 번째 회고

## 25.11.18~21 - 백희연의 첫번째 회고

### 11월 18일

첫 수업인 만큼 오리엔테이션과 개발환경 세팅에 초점을 둔 수업이였다.
vscode의 세팅 자체는 해보았던 경험이 있었지만 직접 독학으로 진행했던 셋팅방식과는 다른 방식과 들어보기만 하였던 약자들의 진짜 의미를 알 수 있었던 시간이였다.

##### 약자 목록

- CS : Computer Science = 컴퓨터 과학 지식
- CLI : Command line interface = 명령어로의 소통
- GUI : Graphic User Interface = 아이콘, 클릭 등의 명력
- NUI : Natural User Interface = 사람의 행동으로 명령, 대표 예시로 아이폰 시리 명령

Shell : 운영체제와 사용자 간의 인터베이스
Window의 경우 Git을 설치하면 Gitbash를 제공하며 Window Power Shell 이 기본으로 내장되어 있다.

MAC의 경우에는 터미널이라는 셀이 내장되어 있고 Zsh을 지원한다.

Shell의 종류는 여러가지가 있고 필요와 선호도에 따라서 선택하여 사용할 수 있다.

운영체제의 종류를 배우면서
**'리누스 토발즈'**라는 개발 인물에 대해서도 알게 되었다.

수업을 듣는 컴퓨터는 윈도우라 윈도우 환경으로 세팅해두었는데
노트북은 맥북을 사용하고 있어 MAC OS 환경도 세팅해 두어 이동 시에도 접근할 수 있도록 세팅해두도록 해야겠다.
